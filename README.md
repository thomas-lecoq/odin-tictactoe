# Odin tic-tac-toe

An intermediate JavaScript project from [The Odin Project](https://www.theodinproject.com/) curriculum: a dynamic tic-tac-toe game. This exercise was an opportunity to practice factory functions and the module pattern.

## Concepts practiced

- Separation of concerns
- Factory functions
- IIFE
- Private variables and functions
- Closures
- Encapsulation

## Lessons learned

Mistakes were made along the way — notes to keep in mind for future projects:

- Project template cleanup should be done at project start and throughout the development phase
- Commit messages should be more readable
- Separation of concerns should be applied more rigorously (different objects in different files, backend/frontend split, accessor/mutator logic clearly separated)
- Branches should be used more often
- Some functions are too complex to read
- Some files contains too many functions
